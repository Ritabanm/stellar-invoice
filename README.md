# 🌌 StellarInvoice

StellarInvoice is a lightweight, premium, single-page client-side web application designed to generate professional PDF invoices instantly. It requires **no accounts, no logins, and zero database storage**—ensuring absolute privacy for your business billing details.

## ✨ Features

- **Split-Screen Workspace**: Edit form fields on the left and see a live, high-fidelity A4 sheet layout rendering instantly on the right.
- **Light & Dark Mode**: Seamless toggle between a premium slate-dark mode and a clean, high-contrast light mode.
- **Local Storage Auto-Save**: Your business/sender info is securely saved to your local browser's storage, automatically pre-filling the fields on your next visit.
- **Dynamic Math Engine**: Instantly computes line sub-totals, custom tax rates, custom discounts, and grand totals on every keystroke.
- **Vector PDF Exports**: Generates crisp, high-definition PDF invoices utilizing browser printing and `html2pdf.js`.
- **100% Private & Serverless**: Runs completely in the browser. Zero client data is ever transmitted to a server.

## 🚀 Getting Started

To run the application locally on your computer:

1. Clone or download this directory.
2. Open `index.html` directly in your web browser, OR start a simple local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Open `http://localhost:8000` in your browser.
